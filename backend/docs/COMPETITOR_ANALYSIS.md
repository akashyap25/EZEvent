## What EZEvent Does Better (USPs Over Competitors)

| Feature | Why It's Better |
|---------|----------------|
| Real-time chat per event | Neither Dreamcast nor Samaaro has in-event chat. Only EZEvent. |
| AI-powered event creation | Generates descriptions, task breakdowns, tags. Unique feature. |
| Recurring events | Built-in with patterns (daily/weekly/monthly). Luma has it, but not Dreamcast/Samaaro. |
| Open-source / self-hosted | All competitors are SaaS-only. EZEvent can be self-hosted. |
| Event collaboration (co-organizers) | Not common in competitors at this depth. |
| Task management per event | Unique — neither Dreamcast nor Samaaro has built-in task assignment. |

## What's Missing (Prioritized Roadmap)

### 🔴 Critical (High impact, users expect it)
1. **Automated event reminders** — Schedule emails 24h and 1h before event (needs cron job / Bull queue)
2. **Multi-tier ticket types** — VIP, General, Speaker, etc. (schema change to tickets array)
3. **Branded event microsite/landing page** — Dedicated URL per event with custom design

### 🟡 High Priority (Differentiators)
4. **Custom registration fields** — Let organizers add custom questions to the registration form
5. **WhatsApp notifications** — Integration with WhatsApp Business API for reminders
6. **Live polls & Q&A** — Real-time audience engagement during events
7. **CRM integration** — HubSpot/Salesforce webhooks for lead sync
8. **Mobile event app** — React Native or PWA for attendees
9. **ROI dashboard** — Revenue vs spend per event/org

### 🟢 Medium Priority (Nice differentiators)
10. **Networking/matching** — Algorithm-based attendee matching for networking events
11. **Post-event survey/feedback** — Auto-send after event ends
12. **Multi-language support** — i18n for Hindi + English
13. **Event feed/timeline** — Social-style updates during events
14. **Engagement scoring** — Track who engaged most (for organizer insights)
15. **AI audience recommendations** — Suggest events based on past behavior

### 🔵 Low Priority (Enterprise only)
16. **Badge printing** — On-site badge generation (hardware dependent)
17. **Facial recognition** — AI-based check-in (privacy concerns)
18. **SSO/SAML** — Enterprise authentication
19. **White-label** — Remove EZEvent branding completely
20. **Self-check-in kiosks** — Tablet-based self-service
